Native Page Transitions Demo

This app is a showcase for the Telerik Native Page Transition Cordova plugin. The plugin supports native transitions like Slide, Flip, Drawer and Curl transitions on various platforms.

If you're using KendoUI to build your apps (this demo app does!), then take a look at the KendoUI adapter. See the Marketplace doc for details.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/WP8 - App package.
